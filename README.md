# LostAndFound

#### Платформа «Бюро находок» для агрегации утерянных и найденных предметов на территории учебного заведения.

## Клонирование и запуск проекта

Проект состоит из 2-х отдельных репозиториев, объединённых с помощью git submodules.
Для полноценного клонирования проекта необходимо воспользоваться командой с флагом рекурсии:
```
git clone --recurse-submodules https://github.com/realjunglebird/LostAndFound.git
```

Для запуска проекта необходим установленный Docker.
В корне проекта нужно выполнить следующую команду:
```
docker-compose up --build -d
```
